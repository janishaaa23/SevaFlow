## 📖 About

SevaFlow is a modern digital governance platform designed to simplify how citizens access government services.

The platform enables users to:

- 📅 Book appointments online
- 🎫 Receive digital queue tokens
- 📍 Track live queue status
- 📄 Upload and verify documents
- 🤖 Get AI-powered assistance
- 🔔 Receive real-time notifications
- 📊 Enable administrators to efficiently manage departments, services, and queues

The long-term vision is to reduce waiting time, improve transparency, and create a seamless digital experience for both citizens and government officials.
